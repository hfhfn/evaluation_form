# 南京AI-02期-智能体应用实战答辩评分表（group09）

> 请大家如实评分，客观评价，选出心中最优秀的团队。

---

## 评分维度

### 维度1：需求分析 + 技术架构 + 可行性分析
> 加分项，如有则计分，无则不计

| 等级 | 分值 |
|------|------|
| 缺失明显或明显应付 | 1 分 |
| 有但不够完整或略粗糙 | 2 分 |
| 三项完整、清晰、合理 | 3 分 |

---

### 维度2：团队分工与协作
> 加分项，如果没有不计分，如果有加分

| 等级 | 分值 |
|------|------|
| 分工模糊或明显一人完成 | 1 分 |
| 有分工，但配合一般 | 2 分 |
| 分工明确，协作顺畅，答辩体现各成员角色 | 3 分 |

---

### 维度3：PPT质量 + 讲解表现

| 等级 | 分值 |
|------|------|
| PPT混乱或讲解卡顿严重 | 1 分 |
| 中规中矩，能讲清楚 | 2 分 |
| 结构清晰，重点突出，讲解流畅自然 | 3 分 |

---

### 维度4：问题发现与解决

| 等级 | 分值 |
|------|------|
| 未提及或问题太假 | 1 分 |
| 有问题但解决过程不清晰 | 2 分 |
| 明确提出真实问题 + 解决思路/过程 | 3 分 |

---

### 维度5：项目完成度

| 等级 | 分值 |
|------|------|
| 功能残缺或演示中途失败 | 1 分 |
| 核心功能完成，有少量 bug 或未完成边缘功能 | 2 分 |
| 功能完整，无明显 bug，可演示全流程 | 3 分 |

---

### 维度6：情感分（团队认同 / 态度 / 氛围）

| 等级 | 分值 |
|------|------|
| 缺乏热情或明显划水 | 0 分 |
| 中规中矩，不反感 | 1 分 |
| 有感染力，让人愿意支持 | 3 分 |
